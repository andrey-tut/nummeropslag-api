# Privacy Policy

## Data processed

The plugin processes only the Danish phone number or business-name query that a
user explicitly provides to a tool. It also requires a Nummeropslag API key,
which Dify stores as a secret provider credential.

The plugin does not read contacts, conversations, files, call logs or unrelated
workspace data. Nummeropslag does not expose names of private individuals.

## Data use and sharing

Requests are sent directly over HTTPS to the first-party Nummeropslag API at
`nummeropslag.dk`. The data is used only to return caller-ID, official CVR,
operator, number-type and aggregated spam/trust results.

No query or credential is shared by the plugin with advertising networks or
other third parties. Service-side processing is governed by the
[Nummeropslag privacy policy](https://nummeropslag.dk/privatlivspolitik).

## Retention

The plugin itself does not persist phone numbers, business queries, API
responses or credentials. Dify workspace logs and Nummeropslag service logs are
subject to their respective administrator settings and published policies.

## Contact

Privacy questions can be sent to `info@lynbro.dk`.
