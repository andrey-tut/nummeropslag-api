# Security

Please report security vulnerabilities privately through
[GitHub Security Advisories](https://github.com/andrey-tut/nummeropslag-api/security/advisories/new).

Do not include API keys, phone numbers, personal information, or exploit details in
public issues. For ordinary bugs and installation support, use the repository's
[issue tracker](https://github.com/andrey-tut/nummeropslag-api/issues).
