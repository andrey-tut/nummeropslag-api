# Nummeropslag MCP

This bundle installs the Nummeropslag MCP server in compatible desktop clients.

During installation, provide a Nummeropslag Partner API key from
https://nummeropslag.dk/api-noegle. The key is stored by the host application and
passed to the local MCP server as an environment variable.

## Privacy Policy

The local server sends only the number or business query requested by the user to
the first-party Nummeropslag API. It does not read conversations, files, contacts,
or call logs. Nummeropslag's data collection, use, retention, sharing, and contact
information are documented at:

https://nummeropslag.dk/privatlivspolitik
